3 Add to Your Theme

Add the code to your Shopify theme:

  1. Go to Online Store > Themes in your Shopify admin
  2. Click Actions > Edit code on your current theme
  3. Find the theme.liquid file in the Layout folder
  4. Paste the code before the closing </body> tag
  5. Click Save
4 Test Your Widget
